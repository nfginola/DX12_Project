# DX12_Project
First project in DX12

* __Techniques__:
	* Vertex Pulling
	* Suballocated Buffers (constant buffers only)
	* Bindless Textures
	* Direct Descriptor Access (requires Agility SDK)
	* Async Copy
	* Multiple frames in flight
	* Inline Raytraced Shadows (Directional light)
